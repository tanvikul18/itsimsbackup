Video requirements

On the social networking page, FriendList, you want to provide videos of the project. Close the Windows Photo Gallery. Open Microsoft Edge, type http://www.friendlist.com/videos and press ENTER to open the webpage that lists file requirements for video uploads to the site.

Read the information and note the acceptable file formats. Close the browser window. In File Explorer, use the back button to return to the Beautification folder. Open the Videos folder, which contains two videos to be uploaded. Based on the guidelines for FriendList, open the video file that you will upload.
