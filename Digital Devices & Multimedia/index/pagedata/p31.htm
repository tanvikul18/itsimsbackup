Click the Next button in Windows Photo Gallery to view the next image. This picture needs to be cropped.

Using the Crop tool, drag the right side of the grid to remove the person on the right side of this image, leaving the tan bucket on the ground to his left. Press ENTER to apply the crop. Then, using the file size information in the right pane, make a copy of the image and save it in the appropriate folder with the same file name.
