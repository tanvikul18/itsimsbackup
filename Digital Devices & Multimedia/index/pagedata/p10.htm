You need to edit and organize the images using Windows Photo Gallery, which has been installed on your computer. Images for the FriendList website should be less than1 MB in size. Images that will be printed in the Newspaper should be greater than 1 MB.

In the Pictures folder, double-click image1_inner-city-decay to open the image in the editor. The file properties are displayed in the right pane. Using the file size information, make a copy of the image and save it in the appropriate folder with the same file name.
